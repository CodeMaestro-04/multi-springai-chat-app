# Multi SpringAI Chat App

A full-stack AI chat application that integrates OpenAI, Ollama, and Anthropic to compare responses from multiple AI models in a single interface.

---

## 🛠️ Tech Stack

- React (Frontend)
- Spring Boot (Backend)
- AI Models: OpenAI, Ollama, Anthropic

---

## ⚙️ Setup & Run

### 1️⃣ Frontend
```bash
npm install
npm run dev


2️⃣ Backend
Run the Spring Boot application from your IDE

3️⃣ Ollama (Local AI)
ollama serve

⚠️ Note
API keys for OpenAI and Anthropic are not included due to security reasons
You can add your own keys in application.properties if needed
Ollama runs locally and does not require any API key

🚀 Features
Compare responses from multiple AI models
Supports both cloud and local AI
Simple and interactive UI
